This Streamlit app maps threats from Microsoft Threat Modeling Tool CSV exports to Alstom PCyA cybersecurity requirements based on IEC 62443-4-2.

## How to use
1. Upload **Threat Model CSV** (from Microsoft TMT).  
2. Upload **PCyA Excel** (requirements list).  
3. Optionally override the default **Candidates CSV** or **IEC 62443 Excel**.  
4. Click **Run** to generate mappings and diagnostics.  
5. Download the mapped CSV for review.
